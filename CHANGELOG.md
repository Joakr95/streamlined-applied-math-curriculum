# Changelog

All notable changes to the Streamlined Applied Math Curriculum will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-19

### Added
- Initial release of Streamlined Applied Math Curriculum
- Complete 57-week curriculum (13-14 months)
- Phase 0: Learning Foundations (1 week)
- Phase 1: Core Mathematics (26 weeks)
  - Multivariable Calculus (8 weeks)
  - Linear Algebra (10 weeks)
  - Probability & Statistics (8 weeks)
- Phase 2: Applied Core (24 weeks)
  - Optimization (10 weeks)
  - Differential Equations (8 weeks)
  - Stochastic Processes (4 weeks)
  - Scientific Computing (2 weeks)
- Phase 3: Computational Mastery (6 weeks)
  - Numerical Linear Algebra
  - Capstone projects
- Six capstone project options covering multiple quantitative fields
- Career outcome information for finance, tech, consulting, economics, insurance
- Study strategy and weekly schedule recommendations
- Complete resource list with verified links
- Completion checklist
- Getting started guide

### Resources
- All resources are free and self-paced
- Courses from MIT, Stanford, Harvard
- Supplementary content from 3Blue1Brown, Khan Academy
- Total cost: $0

### Philosophy
- Essentials-only approach
- Broadly applicable to all quantitative careers
- Realistic time estimates with buffers
- Project-based learning throughout
- Career-focused rather than academic-focused
