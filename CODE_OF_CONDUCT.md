# Code of Conduct

## Our Pledge

We are committed to providing a welcoming and inspiring community for all learners, contributors, and maintainers. We pledge to make participation in this project a harassment-free experience for everyone, regardless of age, body size, disability, ethnicity, gender identity and expression, level of experience, nationality, personal appearance, race, religion, or sexual identity and orientation.

## Scope

This Code of Conduct applies in all project spaces, including:
- GitHub repository (issues, PRs, discussions)
- Related community channels (Discord, Reddit)
- When representing the project in public spaces

## Enforcement

### Reporting
If you experience or witness unacceptable behavior, please report it by:
- Opening a confidential issue
- Emailing the project maintainers
- Reporting through GitHub's reporting features

### Consequences
Maintainers will review and investigate all complaints. Possible actions include:
1. **Warning:** A private warning about the behavior
2. **Temporary Ban:** Temporary restriction from project spaces
3. **Permanent Ban:** Permanent removal from all project spaces

Maintainers have the right and responsibility to remove, edit, or reject comments, commits, code, issues, and other contributions that don't align with this Code of Conduct.
